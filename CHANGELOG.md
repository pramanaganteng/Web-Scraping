## [1.0.0] - 2025-12-27

### Added

- Implementasi frontend menggunakan Next.js 16
- Implementasi backend menggunakan Flask Python
- Fitur web scraping untuk data peminjaman ruang kelas
- Sistem autentikasi dan login
- Export data ke format Excel (.xlsx)
- Responsive design untuk mobile dan desktop
- Database SQLite untuk storage
- API endpoints untuk operasi CRUD
- Protected routes untuk keamanan
- Modal components untuk Notes dan Tags
- Context API untuk state management

### Features

- **Frontend:**
  - Next.js dengan App Router
  - TypeScript untuk type safety
  - Tailwind CSS untuk styling
  - React 19 dengan modern hooks
  - Iconify untuk icon library
- **Backend:**
  - Flask REST API
  - BeautifulSoup4 untuk scraping
  - SQLite database
  - CORS support
  - Request handling
